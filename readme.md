# DIY Golf Simulator Enclosure

<img width="1200" height="673" alt="image" src="https://github.com/user-attachments/assets/6d827ac5-6864-49c3-93d2-9637fbe7c941" />


This repository documents a 12′ × 9′ × 5′ indoor golf simulator enclosure built with SteelTek pipe, blackout fabric, an impact screen, short-throw projector, and a Uneekor EYE MINI CORE launch monitor.

The goal is to record:

- Dimensions and layout
- Parts and sources
- Frame cuts and connectors
- Fabric and screen mounting
- Launch monitor, projector, and hitting mat choices

---

## Overview

- **Internal enclosure size:** 12′ 4″ × 9′ 4″ × 5′
- **Impact screen:** 12′ × 9′
- **Launch monitor:** Uneekor EYE MINI CORE
- **Projector:** Short-throw ceiling/pipe-mounted
- **Hitting mat:** Golfguru dense turf mat

The frame is built with SteelTek pipe and set-screw fittings, with a two-layer tarp + blackout fabric enclosure.

## Impact Screen and Tension System

**Impact Screen:** 12′ × 9′

<img alt="image" src="https://github.com/user-attachments/assets/ddde1026-aaf0-4e17-b2c9-6100d84bf0ac" />

https://www.amazon.com/dp/B0DY8ZLYFQ/ref=twister_B0F76QR7TD?th=1&psc=1


Cable tension system:

- 1/8″ galvanized steel cable
- Cable thimbles
- Cable clamps
- Turnbuckles
- Eye bolts or frame anchor points

Screen attaches to the cable via ball bungees — NOT directly to the pipe.

## Projector Setup

**Projector:** Short-throw Optoma

<img width="933" height="559" alt="image" src="https://github.com/user-attachments/assets/f027b03d-a336-47a0-acba-967097324f6d" />

[Optoma GT2000HDR Compact Theater Projector (Amazon)](https://www.amazon.com/dp/B0C5BBCXRK)

https://region-resource.optoma.com/products/import/Documents/02387d20-4b12-4d04-97e4-be316b803cf2.pdf

Mounting:

- Above and slightly in front of the hitting area
- Height: ~9–10′
- Centered horizontally
- Throw distance chosen based on projector’s throw ratio

This avoids shadows and keeps the ball path unobstructed.
